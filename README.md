# 基于Vue3 + Ts + Vite的低代码平台

## 介绍

本项目参考 https://github.com/woai3c/visual-drag-demo ,是一个基于Vue3 + Typescript + Vite的低代码LowCode平台

感谢谭光志大大的低代码平台开源项目💓

## 安装
npm install

## 运行
npm run dev
